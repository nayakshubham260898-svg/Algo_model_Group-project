# Algo_model_Group-project
This is basic model for algo trading
This model uses the S&P 500 for the analysing and making startegies(refer the file of list of S&P 500)
this model uses rsi, macd bolling bands
For Betas it uses famafrench 3 model and rolling ols
for unsupervised learning it uses the k means clustring.
with efficient frontier it allows us to backtest the model too.

Use auto_adjust=False for getting adjusted close values.
Use Try and except for doing the error handling in the downloading and testing.
